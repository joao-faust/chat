# Chat

Chat with your friends

## Server

```bash
  # Runs the server in development mode
  npm run start:dev

  # Generates the build
  npm run build
```

### Technologies used in server

 - NodeJs;
 - MongoDb;
 - Websockets.

## Client

```bash
  # Runs the client
  python main.py
```

### Tecnologies used in client

 - Python;
 - Websockets.

### Chat commands

 - !help - to show the chat commands;
 - !on - to show all participants;
 - !exit - to exit from the chat.
